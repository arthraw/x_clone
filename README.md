# x_clone
---

Este app consiste em um clone do X focando na parte de login, fiz ele pensando em melhorar minhas habilidades com aplicações android no quesito autenticação.

## Tecnologias:

- UI: *Compose*;
- Interatividade e Backend: *Kotlin*;
- ORM: *ROOM*;
- Arquitetura: *MVVM*
